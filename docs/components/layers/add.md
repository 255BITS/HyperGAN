    # add layer

    Adds two or more layers together. Accepts nested layer definitions.

    ## input size

    Any number of matching tensors

    ## output size

    Same as input size

    ## syntax

    ```json
      "add [layer]*"
    ```

    ## examples

    ```json
      "add self (attention)"
    ```
